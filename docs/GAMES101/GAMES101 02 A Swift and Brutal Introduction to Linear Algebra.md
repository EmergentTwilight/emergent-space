# Vectors

- $\vec{a}$
- $\hat{a}=\vec{a}/||\vec{a}||$
- $\mathbf{a}^T=(x, y)$
- dot product: 求夹角、求投影
- cross product: 
	- 确定一个向量在另一个向量的左侧还是右侧
	- 确定一个点是否在图形内部，如果在所有边的同侧，那么一定在内部
